Sobre este preview: tudo aqui é HTML/CSS puro, sem framework. Os componentes do Next.js vão replicar esses blocos 1:1 usando Tailwind + shadcn com o theme sobrescrito. Passa o mouse sobre os cards pra ver o spring hover. Troca o accent no topo pra ver como cada cor muda o sistema.

Liquid Glass: o efeito depende de backdrop-filter. Safari/Chrome modernos = OK. Firefox < 103 cai num fallback opaco.
